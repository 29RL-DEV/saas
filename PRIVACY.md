## 7. DATA SECURITY

### 7.1 Security Measures

We protect your data with:

| Layer         | Implementation                 |
| ------------- | ------------------------------ |
| Transport     | HTTPS/TLS encryption           |
| Storage       | PostgreSQL database encryption |
| Access        | Role-based access control      |
| Passwords     | PBKDF2 hashing (NIST approved) |
| Backup        | Encrypted daily backups        |
| Monitoring    | Sentry error tracking          |
| Rate Limiting | 1000 req/min DDoS protection   |

### 7.2 Password Security

- ✅ Hashed with PBKDF2 (not reversible)
- ✅ Salted (unique per password)
- ✅ NEVER stored in plain text
- ✅ NEVER logged or displayed
- ✅ Minimum 8 characters required
- ✅ Optional two-factor authentication (coming soon)

### 7.3 What If There's a Breach?

If we discover a data breach:

1. Immediately notify supervisory authority (ICO UK)
2. Notify all affected users within 72 hours
3. Provide details of breach & steps taken
4. Cooperate with investigation
5. Maintain incident log for 3 years

**Your Rights:**

- ✅ Notification within 72 hours
- ✅ Details of breach
- ✅ Recommended protective measures
- ✅ Contact for further information
